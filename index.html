HTML

Hypertext Markup Language

  • HTML elements work together to build the structure of a webpage and provide a framework.
  • The head element contains information about a webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.

CSS

Cascading Style Sheets

  • CSS is used to style the content of a webpage.
  • There are three ways to add CSS to a webpage: inline, internal, and external.
  • Inline styles are added directly to an HTML element.
  • Internal styles are added to the head element of an HTML document.
  • External styles are added to a separate CSS file and linked to an HTML document.
  • Classes are used to style multiple elements on a webpage.
  • IDs are used to style a single element on a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

Version Control System

  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and swithces to it
  • git add .: Adds all files to the staging area
  • git commit -m "message": Commits all files in the staging area
  • git push origin branch-name: Pushes all commits to the remote repository
  • git pull origin branch-name: Pulls all commits from the remote repository
  • git checkout branch-name: Switches to an existing branch
  • git branch: Lists all branches
  • git branch -d branch-name: Deletes a branch

JavaScript

Programming Language

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are reusable blocks of code that perform a specific task.
  • Arguments are the actual data that's passed into a function.
  • Return statements allow us to output data from a function.
  • Arrays are lists of data that are stored in a single variable.
  • Loops allow us to execute a block of code multiple times.
  • Conditional statements allow us to execute a block of code only if a certain condition is met.
