Git

  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and swithces to it
  • git add .: Adds all files to the staging area
  • git commit -m "message": Commits all files in the staging area
  • git push origin branch-name: Pushes all commits to the remote repository
  • git pull origin branch-name: Pulls all commits from the remote repository
  • git checkout branch-name: Switches to an existing branch
  • git branch: Lists all branches
  • git branch -d branch-name: Deletes a branch
